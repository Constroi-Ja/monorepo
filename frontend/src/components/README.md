# Components

Este diretório contém componentes reutilizáveis da aplicação.

## Estrutura

- `ui/` - Componentes de interface básicos (Button, Input, Card, etc.)
- `layout/` - Componentes de layout (Header, Footer, Sidebar, etc.)
- `features/` - Componentes específicos de funcionalidades

## Convenções

- Use TypeScript para todos os componentes
- Componentes devem ser funcionais com hooks
- Use nomes descritivos e em PascalCase
- Exporte componentes como default quando possível
- Documente props com TypeScript interfaces
